Homebrew 이용해 git 설치

Jun 11, 2015 | environment

새로 맥북을 사서 git을 설치하기 전에 확인을 위해 터미널에서

뒤에 Apple Git-55라는 매우 수상한 문자가 붙어있어 검색을 해보았더니, Mac은 Apple 전용인 오래된 버전의 Git을 기본적으로 가지고 있다고 한다. 오래된 버전이라는 것은 둘째치고 Apple전용이라는 말에 찝찝하여 Git을 새로 설치했다.

일단 Homebrew 명령어를 이용해 Git을 설치한다.
