이클립스에서 웹 프로젝트를 실행하면 기본적으로는 이클립스 내장 브라우저를 사용한다. 그게 싫고 시스템 기본 브라우저를 사용하고 싶다면!
```
Eclipse Menu > Window > Web Browser > Default system web browser
```
