블로그에 글 쓰기도 부끄러울 만큼 쉽다...
Homebrew가 설치되었다는 가정 하에!

```shell
brew install scala
```

그리고 설치가 다 된 후 scala를 입력해보면 scala REPL이 나타나면서 바로바로 테스트를 해볼 수 있다.
REPL에서 벗어나오려면 Control + C (Mac에서)
